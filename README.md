# Cote

Marketing site for Cote (cote.studio). Static HTML/CSS/JS, no build step.

- `index.html` — the page
- `styles.css` — styles (Whenevr-style: gray canvas, white cards, serif italic accents)
- `script.js` — theme toggle, nav scroll state, scroll reveals

Deployed on Vercel. Prospect sites live at `/{business-slug}/index.html`.
